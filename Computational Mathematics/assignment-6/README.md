# Формулы численных методов решения ОДУ

## 1. Метод Эйлера (Euler's Method)

```
yᵢ₊₁ = yᵢ + h·f(xᵢ, yᵢ)
xᵢ₊₁ = xᵢ + h
```

## 2. Модифицированный метод Эйлера (Modified Euler's Method)

```
y* = yᵢ + h·f(xᵢ, yᵢ)
yᵢ₊₁ = yᵢ + h·[f(xᵢ, yᵢ) + f(xᵢ₊₁, y*)]/2
xᵢ₊₁ = xᵢ + h
```

## 3. Метод Рунге-Кутты 4-го порядка (Runge-Kutta RK4)

```
k₁ = h·f(xᵢ, yᵢ)
k₂ = h·f(xᵢ + h/2, yᵢ + k₁/2)
k₃ = h·f(xᵢ + h/2, yᵢ + k₂/2)
k₄ = h·f(xᵢ + h, yᵢ + k₃)

yᵢ₊₁ = yᵢ + (k₁ + 2k₂ + 2k₃ + k₄)/6
xᵢ₊₁ = xᵢ + h
```

## 4. Метод Тейлора (Taylor's Method)

```
y(x*) = y₀ + (x*-x₀)·y'₀ + (x*-x₀)²/2!·y''₀ + (x*-x₀)³/3!·y'''₀ + ... + (x*-x₀)ᴺ/N!·y⁽ᴺ⁾₀
```

Производные для y' = f(x, y):
```
y' = f(x, y)
y'' = ∂f/∂x + ∂f/∂y·f(x, y)
y''' = ∂²f/∂x² + 2·∂²f/∂x∂y·f + ∂f/∂y·y''
```

## 5. Метод Пикара (Picard's Method)

```
φ₀(x) = y₀
φₙ₊₁(x) = y₀ + ∫[x₀ to x] f(t, φₙ(t)) dt
```

---

## Порядок точности

- Эйлер: O(h)
- Модифицированный Эйлер: O(h²)
- Рунге-Кутта RK4: O(h⁴)
- Тейлор: O(hᴺ⁺¹)
- Пикар: зависит от количества итераций
