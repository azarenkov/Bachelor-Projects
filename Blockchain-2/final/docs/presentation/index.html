Protocol° Scenario A · DeFi Super-App
What is this protocol

A miniature Aave + Uniswap + MakerDAO, glued by a DAO.

Trade
SimpleAMM

Constant-product (x·y=k) pair with a 0.3% fee. LPs deposit liquidity, traders pay the spread. Built from scratch, not forked.

Earn
YieldVault

ERC-4626 share vault. Deposit dUSDC → receive vdUSDC. 10% performance fee skim to the DAO Treasury on profit.

Govern
DAO

OpenZeppelin Governor + 2-day Timelock + ERC20Votes token. Every parameter change is on-chain, voted, time-locked.

Every required component from the syllabus is implemented as a first-class concern, wired together end-to-end and demonstrable live on a public L2.

Protocol° Math · AMM

x · y = k, nothing more.

Spot pricing
// reserveOut * amountInWithFee / (reserveIn + amountInWithFee) function _getAmountOut( uint256 amountIn, uint112 reserveIn, uint112 reserveOut ) internal pure returns (uint256) { uint256 fee = amountIn * 997; // 0.3% pool fee uint256 num = fee * reserveOut; uint256 den = reserveIn * 1000 + fee; return num / den; }
k-invariant test
function testFuzz_swap_kNeverDecreases( uint96 amountIn, bool zeroForOne ) public { (uint112 r0a, uint112 r1a, ) = amm.getReserves(); uint256 kBefore = uint256(r0a) * uint256(r1a); amm.swapExactIn(tokenIn, amountIn, 0, alice); (uint112 r0b, uint112 r1b, ) = amm.getReserves(); assertGe(uint256(r0b) * uint256(r1b), kBefore); }

Property tested with 256 random inputs per run. Augmented by a stateful invariant that asserts reserves match the contract's ERC-20 balances after any sequence of add/remove/swap calls.

Protocol° Security · Slither + audit

Defenses inventory
  • Checks-Effects-Interactions on every AMM / Vault entry point, plus nonReentrant defence-in-depth.
  • Every privileged function gated by OpenZeppelin AccessControl; no Ownable, no tx.origin.
  • All ERC-20 calls use SafeERC20. All ETH sends use call{value:}.
  • UUPS upgrade gated by Timelock-only UPGRADER_ROLE; 50-slot storage gap reserved for V2+.
  • Oracle reverts on stale price (configurable per asset) and on non-positive answer.
  • AMM Pausable circuit breaker — Timelock-controlled.
  • ERC-4626 decimal-offset 6 mitigates first-depositor inflation attack.
